Tentang LED Community

LED Community atau Lascar Engineering Developer Community berdiri pada tahun 2019. Merupakan suatu komunitas di SMK PGRI Telagasari yang berfokus pada bidang Development
Terdapat beberapa pembagian tim di komunitas ini yaitu :

Front-end developers bertanggung jawab untuk mengembangkan tampilan dan interaksi pengguna pada website atau aplikasi, fokus pada aspek visual dan fungsional yang dapat dilihat oleh pengguna akhir. Mereka menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript untuk menciptakan antarmuka pengguna yang responsif dan menarik.

Seorang web developer bertanggung jawab untuk merancang, Web developers, di sisi lain, berfokus pada pengembangan backend dari sebuah website atau aplikasi. Mereka bekerja dengan server, database, dan aplikasi logika untuk memastikan bahwa website atau aplikasi berfungsi dengan lancar. Mereka dapat menggunakan berbagai bahasa pemrograman seperti Python, Ruby, atau Java untuk membangun dan memelihara infrastruktur teknis yang mendukung fungsi keseluruhan dari sebuah proyek.

UI designers, atau desainer antarmuka pengguna, bertanggung jawab untuk merancang elemen visual dari website atau aplikasi agar dapat memberikan pengalaman pengguna yang optimal. Mereka memperhatikan estetika, kegunaan, dan konsistensi desain, sering kali menggunakan alat seperti Adobe XD atau Sketch untuk membuat prototipe desain dan bekerja sama dengan tim pengembang untuk mengimplementasikannya.

Tim kreatif, di sisi lain, mungkin terdiri dari berbagai peran seperti desainer grafis, penulis konten, dan spesialis pemasaran. Mereka bekerja bersama untuk menghasilkan ide-ide kreatif dan konten yang menarik untuk mempromosikan produk atau layanan. Tim kreatif dapat terlibat dalam pengembangan merek, kampanye iklan, dan strategi pemasaran secara keseluruhan.
